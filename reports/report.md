# Отчет по проекту: Навигация дифференциального привода с использованием Reinforcement Learning

## Введение

В данном проекте реализована задача навигации робота с дифференциальным приводом в плоскости с использованием алгоритмов reinforcement learning (RL). Робот спавнится в случайной позиции в пределах [-10, 10] по координатам x и y, с случайным углом поворота. Цель - достичь точки (0, 0) с углом 0.

## Описание среды

- **Состояние**: [x, y, θ] - позиция и угол робота
- **Действия**: [v_left, v_right] - скорости левых и правых колес (в диапазоне [-1, 1])
- **Награда**: -distance - 0.1 * |angle_diff|, где distance - расстояние до цели, angle_diff - разница углов
- **Цель**: distance < 0.1 и |angle_diff| < 0.1

## Алгоритм RL

Используется алгоритм Soft Actor-Critic (SAC) из библиотеки Stable Baselines3. SAC подходит для задач с непрерывными действиями и обеспечивает стабильное обучение.

### Параметры обучения:
- Total timesteps: 100,000
- Policy: MLP
- Learning rate: default

## Реализация

### Код структуры:
- `environment.py`: Определение среды Gymnasium
- `train.py`: Обучение агента
- `visualize.py`: Визуализация на Pygame

### Зависимости:
- gymnasium
- stable-baselines3
- pygame
- numpy
- torch

## Результаты

После обучения агент способен успешно достигать цели из случайных начальных позиций.

## Визуализация

Анимация показывает движение робота в реальном времени с использованием Pygame. Робот изображен синим кругом, цель - красным кругом, направление - черной линией.

## Заключение

Проект демонстрирует применение RL для решения задачи навигации мобильного робота. SAC показал хорошую производительность для данной задачи.
